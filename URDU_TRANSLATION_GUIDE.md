# Urdu Translation Guide

## Current Status
The language switcher has been implemented and the folder structure for Urdu translations is in place. When you switch to Urdu, you'll still see English content because the actual Urdu translations haven't been added yet.

## Required Steps for Full Urdu Support

### 1. Translate Document Content
Each document file in the English docs needs to be individually translated to Urdu:

#### Example translation process:
1. Open the English document: `docs/part1-foundations/intro.md`
2. Translate it to Urdu and save with the same filename in the Urdu directory
3. The translated file should be: `i18n/ur/docusaurus-plugin-content-docs/current/part1-foundations/intro.md`

### 2. Update YAML Frontmatter
The frontmatter of each document needs to be updated with Urdu titles and metadata:

English version:
```yaml
---
title: Introduction to Humanoid Robotics
sidebar_label: Introduction
slug: /
description: Learn about humanoid robotics and physical AI
---
```

Urdu version:
```yaml
---
title: ہیومنوڈ روبوٹکس کا تعارف
sidebar_label: تعارف
slug: /
description: ہیومنوڈ روبوٹکس اور فزیکل AI کے بارے میں سیکھیں
---
```

### 3. How to Add Urdu Translations

Since translating an entire book would be extensive, here's how to begin:

1. Pick one document at a time to translate
2. Open the English version from `docs/[section]/[file].md`
3. Open the corresponding Urdu file in `i18n/ur/docusaurus-plugin-content-docs/current/[section]/[file].md`
4. Translate the content while maintaining the same structure and frontmatter format
5. Repeat for each document

### 4. Example Translation
As a demonstration, here's how you would translate the intro file:

English (`docs/part1-foundations/intro.md`):
```markdown
---
title: Introduction to Humanoid Robotics
sidebar_label: Introduction
slug: /
description: Learn about humanoid robotics and physical AI
---

# Introduction to Humanoid Robotics

Welcome to the world of humanoid robotics...
```

Urdu (`i18n/ur/docusaurus-plugin-content-docs/current/part1-foundations/intro.md`):
```markdown
---
title: ہیومنوڈ روبوٹکس کا تعارف
sidebar_label: تعارف
slug: /
description: ہیومنوڈ روبوٹکس اور فزیکل AI کے بارے میں سیکھیں
---

# ہیومنوڈ روبوٹکس کا تعارف

ہیومنوڈ روبوٹکس کی دنیا میں خوش آمدید...
```

## Important Notes
- The file names must remain identical between English and Urdu versions
- Only when Urdu files exist will the system show Urdu content
- If an Urdu file doesn't exist, it will fallback to the English version automatically
- The RTL (right-to-left) styling for Urdu is already configured in the docusaurus.config.ts
- Since the book uses auto-generated sidebars, the sidebar navigation will show Urdu titles only after the individual document files are translated

## Sidebar Navigation
Since the sidebars.ts uses auto-generated content (`type: 'autogenerated'`), the sidebar titles will reflect the titles in the individual document files. So translating the document files will also update the sidebar navigation in Urdu.

The language switcher is now properly configured and functional - you just need to translate the actual content for it to appear in Urdu.
