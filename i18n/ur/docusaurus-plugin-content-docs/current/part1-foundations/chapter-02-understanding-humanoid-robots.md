# ہیومنوڈ روبوٹس کو سمجھنا

**Humanoid Robots** (ہیومنوڈ روبوٹس) وہ روبوٹس ہیں جو انسانوں کی طرح شکل رکھتے ہیں۔ یہ روبوٹس انسانوں کی طرح چل سکتے ہیں، بات کر سکتے ہیں، اور کام کر سکتے ہیں۔

یہ روبوٹس انسانوں کی طرح حرکت کرنے کے لیے خاص طریقے استعمال کرتے ہیں۔
